### 📗스터디 진행 방식

- 주마다 알고리즘 개념을 정하고 관련 문제 다섯 개 풀기
- 매주 화요일 저녁 20 ~ 21시 코드 리뷰 진행 
- 문제는 백준 추천문제를 참고하여 선정 (https://github.com/tony9402/baekjoon)
- 10월 이후 스터디는 프로그래머스의 문제로 진행
- Mattermost & Discord 활용
